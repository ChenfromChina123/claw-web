这是一个基于 **Vue 3 + Element Plus + Monaco Editor** 的完整 IDE 风格前端架构方案。采用模块化组件设计，支持四区域独立拖拽缩放与折叠。

---

### 🛠️ 技术栈选型
| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.4+ | 核心框架 (Composition API) |
| Element Plus | 2.5+ | UI 组件库 (布局/树状图/按钮) |
| Monaco Editor | 0.45+ | 代码编辑器内核 (VS Code 同款) |
| Splitpanes | 3.1+ | 可拖拽分割面板 |
| Pinia (可选) | 2.1+ | 状态管理 (大型项目推荐) |

---

### 📁 项目结构
```
src/
├── components/
│   ├── IDELayout.vue        # 主布局容器
```

### 💻 核心代码实现

#### 1. 主布局组件 (`IDELayout.vue`)
使用 `splitpanes` 实现可拖拽分割，四个区域均可独立折叠：

```html
<template>
  <div class="ide-container">
    <!-- 顶部工具栏 -->
    <div class="toolbar">
      <el-button-group>
        <el-button :icon="FolderOpened" @click="toggleLeftPane">文件</el-button>
        <el-button :icon="ChatDotRound" @click="toggleRightPane">Agent</el-button>
      </el-button-group>
    </div>

    <!-- 主体分割区域 -->
    <splitpanes class="default-theme" horizontal>
      <!-- 左侧文件管理器 -->
      <pane :size="leftSize" :min-size="20" collapsible>
        <div class="pane-header">
          <span>文件资源管理器</span>
          <el-button link @click="toggleLeftPane">
            <el-icon><DArrowLeft /></el-icon>
          </el-button>
        </div>
        <FileExplorer />
      </pane>

      <!-- 中间编辑区 (不可完全折叠) -->
      <pane :size="centerSize" :min-size="30">
        <CodeEditor />
      </pane>

      <!-- 右侧面板 -->
      <pane :size="rightSize" :min-size="20" collapsible>
        <div class="pane-header">
          <span>AI Assistant</span>
          <el-button link @click="toggleRightPane">
            <el-icon><DArrowRight /></el-icon>
          </el-button>
        </div>
        
        <!-- 右侧上下分割 -->
        <splitpanes vertical>
          <pane :size="70">
            <AgentPanel />
          </pane>
          <pane :size="30">
            <SessionManager />
          </pane>
        </splitpanes>
      </pane>
    </splitpanes>
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { Splitpanes, Pane } from 'splitpanes'
import { FolderOpened, ChatDotRound, DArrowLeft, DArrowRight } from '@element-plus/icons-vue'
import FileExplorer from './FileExplorer.vue'
import CodeEditor from './CodeEditor.vue'
import AgentPanel from './AgentPanel.vue'
import SessionManager from './SessionManager.vue'
import 'splitpanes/dist/splitpanes.css'

// 面板尺寸状态
const leftSize = ref(20)
const centerSize = ref(60)
const rightSize = ref(20)

// 折叠切换逻辑
const toggleLeftPane = () => {
  leftSize.value = leftSize.value === 0 ? 20 : 0
}
const toggleRightPane = () => {
  rightSize.value = rightSize.value === 0 ? 20 : 0
}
</script>

<style scoped>
.ide-container {
  height: 100vh;
  display: flex;
  flex-direction: column;
  background: #1e1e1e;
  color: #ccc;
}
.toolbar {
  padding: 8px 16px;
  background: #252526;
  border-bottom: 1px solid #333;
}
.pane-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 8px 12px;
  background: #2d2d2d;
  font-size: 13px;
  border-bottom: 1px solid #333;
}
:deep(.splitpanes__splitter) {
  background: #333;
}
</style>
```

#### 2. 代码编辑器组件 (`CodeEditor.vue`)
集成 Monaco Editor，支持语法高亮：

```html
<template>
  <div ref="editorContainer" class="editor-container"></div>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
import * as monaco from 'monaco-editor'
import editorWorker from 'monaco-editor/esm/vs/editor/editor.worker?worker'
import jsonWorker from 'monaco-editor/esm/vs/language/json/json.worker?worker'

// 配置 Web Workers
self.MonacoEnvironment = {
  getWorker(_, label) {
    if (label === 'json') return new jsonWorker()
    return new editorWorker()
  }
}

const editorContainer = ref(null)
let editor = null

onMounted(() => {
  editor = monaco.editor.create(editorContainer.value, {
    value: `// 欢迎使用 AI IDE
function hello() {
  console.log("Hello, World!");
}`,
    language: 'javascript',
    theme: 'vs-dark',
    automaticLayout: true,
    minimap: { enabled: false }
  })
})

onUnmounted(() => {
  editor?.dispose()
})
</script>

<style scoped>
.editor-container {
  height: 100%;
  width: 100%;
}
</style>
```

#### 3. 文件管理器 (`FileExplorer.vue`)
使用 Element Plus 树形组件：

```html
<template>
  <el-tree
    :data="fileTree"
    :props="defaultProps"
    node-key="id"
    default-expand-all
    highlight-current
    style="color: #ccc;"
  >
    <template #default="{ node, data }">
      <span class="custom-tree-node">
        <el-icon><component :is="data.icon" /></el-icon>
        <span>{{ node.label }}</span>
      </span>
    </template>
  </el-tree>
</template>

<script setup>
import { ref } from 'vue'
import { Folder, Document, Code } from '@element-plus/icons-vue'

const defaultProps = { children: 'children', label: 'label' }
const fileTree = ref([
  {
    id: 1, label: 'src', icon: Folder,
    children: [
      { id: 4, label: 'App.vue', icon: Code },
      { id: 5, label: 'main.js', icon: Document }
    ]
  },
  { id: 2, label: 'index.html', icon: Document },
  { id: 3, label: 'package.json', icon: Document }
])
</script>

<style scoped>
.custom-tree-node {
  display: flex;
  align-items: center;
  gap: 8px;
  font-size: 14px;
}
:deep(.el-tree-node__content:hover) {
  background: #2a2d2e;
}
</style>
```

### 🚀 快速启动配置

#### `package.json` 依赖
```json
{
  "dependencies": {
    "vue": "^3.4.0",
    "element-plus": "^2.5.0",
    "monaco-editor": "^0.45.0",
    "splitpanes": "^3.1.0"
  },
  "devDependencies": {
    "@vitejs/plugin-vue": "^5.0.0",
    "vite": "^5.0.0"
  }
}
```

#### `vite.config.js`
```js
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  server: { port: 3000 }
})
```

### ✨ 核心功能亮点
1. **四区域独立布局**：文件树/编辑区/对话区/会话区均可自由拖拽

3. **极简折叠交互**：点击箭头或工具栏按钮即可隐藏面板
