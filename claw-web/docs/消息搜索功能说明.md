# 消息搜索功能说明

## 功能概述

消息搜索功能允许用户快速搜索历史聊天记录，支持多种筛选条件，帮助用户快速定位所需内容。

## 功能特性

### 1. 搜索条件

- **关键词搜索**: 支持在消息内容中搜索关键词
- **会话筛选**: 可以指定在某个特定会话中搜索
- **日期范围**: 支持按时间范围筛选消息
- **组合搜索**: 支持多个条件组合搜索

### 2. 搜索方式

#### 快捷键方式
- 按下 `Ctrl + F` (Windows) 或 `Cmd + F` (Mac) 快速打开搜索弹窗

#### 按钮方式
- 点击聊天界面顶部工具栏的搜索图标（放大镜图标）

### 3. 搜索结果

- 显示匹配的消息数量
- 每条结果显示：
  - 消息角色（我/助手）
  - 所属会话标题
  - 消息时间
  - 消息内容（关键词高亮显示）
- 点击结果可快速跳转到对应会话和消息

## 使用方法

### 基本搜索

1. 打开搜索弹窗（Ctrl+F 或点击搜索按钮）
2. 在搜索框中输入关键词
3. 点击"搜索"按钮或按回车键
4. 查看搜索结果列表

### 高级搜索

1. 打开搜索弹窗
2. 输入关键词（可选）
3. 选择特定会话（可选，不选则搜索所有会话）
4. 选择日期范围（可选）
5. 点击"搜索"按钮

### 清空搜索

- 点击"清空"按钮可清空所有搜索条件

## 技术实现

### 后端实现

#### API 接口

```
GET /api/sessions/messages/search
```

**请求参数:**
- `keyword` (可选): 搜索关键词
- `sessionId` (可选): 会话 ID
- `startDate` (可选): 开始日期（ISO 格式）
- `endDate` (可选): 结束日期（ISO 格式）
- `limit` (可选): 返回数量限制，默认 100
- `offset` (可选): 偏移量，默认 0

**响应格式:**
```json
{
  "success": true,
  "data": {
    "results": [
      {
        "message": {
          "id": "消息 ID",
          "sessionId": "会话 ID",
          "role": "user|assistant",
          "content": "消息内容",
          "createdAt": "创建时间"
        },
        "sessionTitle": "会话标题",
        "total": 总结果数
      }
    ],
    "total": 总结果数
  }
}
```

#### 数据库查询

使用 MySQL 的 LIKE 操作进行模糊搜索，支持：
- 关键词模糊匹配
- 会话过滤
- 日期范围过滤
- 按创建时间倒序排序

### 前端实现

#### 组件结构

- `MessageSearch.vue`: 搜索弹窗组件
- `Chat.vue`: 主聊天页面，集成搜索功能

#### 功能特性

- 响应式搜索表单
- 实时搜索状态显示
- 关键词高亮显示
- 结果分页（预留）
- 空状态提示
- 加载状态指示

## 注意事项

1. **至少需要一个搜索条件**: 不能进行无条件搜索
2. **关键词高亮**: 仅对字符串类型的消息内容生效
3. **数组类型内容**: 对于复杂内容（如工具调用），显示 JSON 字符串的前 200 个字符
4. **性能考虑**: 默认限制返回 100 条结果，避免一次性加载过多数据
