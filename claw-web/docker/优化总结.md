## 🚀 使用方法

### 方法一：自动配置（推荐）

```powershell
# 1. 以管理员身份运行 PowerShell
cd d:\Users\Administrator\AistudyProject\HAHA\claw-web\docker

# 2. 运行配置脚本
.\configure-docker-mirror.ps1

# 3. 重启 Docker Desktop 后构建
docker compose -f docker-compose.cn-fast.yml up -d --build
```

### 方法二：手动配置 Docker Desktop

1. 打开 Docker Desktop → Settings → Docker Engine
2. 添加以下配置：

```json
{
  "registry-mirrors": [
    "https://docker.mirrors.ustc.edu.cn",
    "https://registry.docker-cn.com",
    "https://hub-mirror.c.163.com"
  ],
  "builder": {
    "gc": {
      "defaultKeepStorage": "20GB",
      "enabled": true
    }
  },
  "experimental": false
}
```

3. 点击 Apply & Restart

### 方法三：手动拉取镜像

```bash
# 拉取基础镜像
docker pull registry.cn-hangzhou.aliyuncs.com/library/node:20-alpine
docker pull registry.cn-hangzhou.aliyuncs.com/library/nginx:1.25-alpine
docker pull ccr.ccs.tencentyun.com/oven/bun:1

# 重命名标签
docker tag registry.cn-hangzhou.aliyuncs.com/library/node:20-alpine node:20-alpine
docker tag registry.cn-hangzhou.aliyuncs.com/library/nginx:1.25-alpine nginx:1.25-alpine
docker tag ccr.ccs.tencentyun.com/oven/bun:1 oven/bun:1

# 构建项目
docker compose -f docker-compose.cn-fast.yml up -d --build
```

1. **科大镜像** (最稳定) - `https://docker.mirrors.ustc.edu.cn`

在 `.env` 文件中可以配置：

```bash

DOCKER_BUILDKIT=1
COMPOSE_DOCKER_CLI_BUILD=1

# 镜像源配置（可选）
MIRROR_SOURCE=aliyun  # aliyun, ustc, netease, tencent
```

## 📞 故障排查

### 常见问题

#### 1. 构建卡在拉取镜像

**症状**: `=> [internal] load metadata for docker.io/...`

**解决**:
```bash
# 手动拉取
docker pull ccr.ccs.tencentyun.com/oven/bun:1
docker tag ccr.ccs.tencentyun.com/oven/bun:1 oven/bun:1

# 重新构建
docker compose -f docker-compose.cn-fast.yml build
```

#### 2. SSL 证书错误

**症状**: `x509: certificate is not valid`

**解决**: 使用 HTTP 协议
```dockerfile
RUN sed -i 's|https://mirrors.aliyun.com|http://mirrors.aliyun.com|g' ...
```

#### 3. 镜像标签不匹配

**症状**: `manifest not found`

**解决**:
```bash
docker pull registry.cn-hangzhou.aliyuncs.com/library/node:20-alpine
docker tag registry.cn-hangzhou.aliyuncs.com/library/node:20-alpine node:20-alpine
```

### 验证配置

```bash
# 检查镜像加速器

# 测试镜像源
curl -I https://docker.mirrors.ustc.edu.cn

# 查看构建日志
```
