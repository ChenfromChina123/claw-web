# 页面高度固定修复计划

## 问题分析
当前布局存在高度约束链断裂的问题：
1. `App.vue` 中 `.app-main` 设置了 `overflow-y: auto`，导致整个页面滚动
2. 但 `Chat.vue` 内部又希望有自己的独立滚动
3. 布局层级间的高度传递不完整，导致 flex 拉伸失效

## 修复目标
- 建立完整的高度约束链
- 确保每个层级都有明确的高度或 min-height: 0
- 实现内部滚动而非整页滚动

## 关键布局原则（修复依据）
1. **Flex 布局高度约束链**: 每个容器必须有明确的高度或 min-height: 0
2. **避免多层 overflow**: 只在最内层需要滚动的地方设置 overflow
3. **显式高度链路**: 从根到叶，每个层级都参与 flex 拉伸
4. **输入框定位**: 固定定位的输入框需要对应的 padding-bottom 预留空间
