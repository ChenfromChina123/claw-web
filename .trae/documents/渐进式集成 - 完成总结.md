## 关键功能汇总

### 1. 性能监控
- **指标记录**: `record(endpoint, duration, success, metadata)`
- **统计报告**: P50/P95/P99/平均值/错误率
- **性能告警**: 自动检测高延迟和高错误率
- **监控端点**:
  - `command.execute` - 命令执行
  - `tool.execute` - 工具执行
  - `mcp.addServer` - 添加 MCP 服务器
  - `mcp.removeServer` - 移除 MCP 服务器
  - `mcp.callTool` - 调用 MCP 工具
  - `mcp.listTools` - 列出工具

### 2. Feature Flag 双轨运行
- **灰度发布**: 支持 0-100% 流量控制
- **环境变量**: 支持运行时动态切换
- **自动回滚**: 每个模块独立切换
- **已配置特性**:
  - `mcp.new.gateway` - MCP 网关切换
  - `commands.hybrid.processor` - 命令处理器
  - `tools.new.executor` - 工具执行器
  - `oauth.enhanced.flow` - OAuth 认证

### 3. 命令历史记录
- **存储限制**: 最多 100 条记录
- **按会话过滤**: 支持 sessionId 过滤
- **记录内容**: 命令、时间戳、sessionId、结果
- **API**:
  - `getHistory(sessionId?, limit?)`
  - `clearHistory(sessionId?)`

### 4. 工具审计日志
- **存储限制**: 最多 1000 条记录
- **按会话过滤**: 支持 sessionId 过滤
- **记录内容**: 工具名、输入、结果、持续时间、sessionId、时间戳
- **API**: `getAuditLog(sessionId?, limit?)`

### 5. 工具限流
- **限流策略**: 每分钟 100 次调用
- **按工具限流**: 每个工具独立计数
- **自动重置**: 每分钟自动重置计数器
- **错误返回**: 超限时返回明确的错误信息
